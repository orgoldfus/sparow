# MVP Scope

The MVP focuses on the **core query workflow**.

Supported database:

- PostgreSQL

Core features:

Connection
- connect to PostgreSQL
- manage saved connections
- secure credential storage

Schema browser
- explore schemas
- tables
- views
- columns
- indexes

SQL editor
- syntax highlighting
- autocomplete
- multiple tabs
- query execution

Results viewer
- fast table rendering
- large result sets
- sorting
- filtering
- export to CSV

Developer productivity

- query history
- saved queries
- keyboard shortcuts
- command palette

The MVP should already feel **fast and pleasant to use**.
